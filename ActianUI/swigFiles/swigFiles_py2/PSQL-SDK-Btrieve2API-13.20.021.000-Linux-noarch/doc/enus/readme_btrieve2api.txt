Welcome to the Btrieve 2 API!

Summary
-------
Btrieve 2 API is a new API for Actian PSQL.  To learn more see the Actian
PSQL Data Access Methods documentation at docs.actian.com.

What is SWIG?
-------------
SWIG is a software development tool that connects programs written in
C++ with a variety of high-level programming languages.  SWIG is used
with different types of target languages, including common scripting
languages such as Perl, PHP and Python.  To learn more about SWIG, visit
www.swig.org.  To understand how Btrieve 2 API works with SWIG, see the
Actian PSQL Data Access Methods documentation at docs.actian.com.
