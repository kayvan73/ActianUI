Btrieve API へようこそ!

まとめ
-------
Btrieve 2 API は Actian PSQL の新しい API です。  詳細については、Actian PSQL の
データ アクセス方法に関するドキュメント（docs.actian.com）を参照してください。

SWIG とは
-------------
SWIG は、C++ および様々な高級プログラミング言語で書かれたプログラムを結合するソフトウェア
開発ツールです。  SWIG は、Perl、PHP、Python などの代表的なスクリプト言語を含む、様々な
種類の目的言語と共に使用できます。  SWIG の詳細については、www.swig.org を参照して
ください。  Btrieve 2 API と SWIG の連携の仕組みについては、Actian PSQL のデータ アクセス
方法に関するドキュメント（docs.actian.com）を参照してください。
